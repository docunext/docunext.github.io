Setup Wifi on HP Laser Color Printer 150nw

July 14th, 2020

In case someone else finds it difficult to setup wifi (or the printer in general), here are some notes from my experience.

The default username and password for my printer was "admin" with no password. You may like to change it. I did.

I was unable to get the web interface to work when using Chrome or Firefox. I accessed the interface when I connected the printer to my network with an ethernet cable.

I tried two actions and both failed:

  • setting up wifi
  • updating the admin password

I had a hunch it wasn't my problem, so I tried Internet Explorer. That was it

  • after that everything was easy.
