Iceweasel needs your help!

There are many ways non-developers can help improve Iceweasel:

  • Draw a new logo,
  • Report bugs,
  • Help triaging existing bugs
    • Check if they still exist,
    • Find duplicate bugs,
    • If they are reproducible,
    • If they are Iceweasel-only or apply to Firefox as well,
    • If they are reported on Firefox bug tracker,
  • Enhance documentation,
  • etc.

There are many ways developers can help, too ;).

In all cases, you may want to contact the pkg-mozilla-maintainers list.
