I'm installing NoMachine FreeNX:

Download 3 debs from nomachine.com, then install

aptitude install libstdc++2.10-glibc2.2

dpkg -i nxclient...

dpkg -i nxnode...

dpkg -i nxserver...

Then use the no machine client to connect. Great product!
