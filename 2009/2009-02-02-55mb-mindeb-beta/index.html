55MB Mindeb Beta

55MB Mindeb Beta

Root password is nodows, can run with this command:

qemu -m 128 -vnc :0 -net nic,model=rtl8139 -net tap -hda nodows.img

(The archive is bzip2 compressed, and cannot run prior to being decompressed.)
