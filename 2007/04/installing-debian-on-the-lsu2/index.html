Installing Debian on the NLSU2

April 26th, 2007

Today I installed debian on NLSU2. An NLSU2 is a device that lets you access a hard drive over a network. We had some issues attempting to install Debian. At first we suspected it was the fact that we removed the jumper block which limits the data transfer to 1.5 gbits per second. After opening up the drive and putting the block back on, we discovered that this wasn't the case. Upon further investigation we learned that the 400GB hard drive was too big, so we ended up using a 60gb. Luckily after this, the installation worked. All in all, the whole process took around an hour to complete.

Problems

  1. The only problem we really ran into was the fact that the initial hard drive we were using was too large, and the partitioner would freeze at 33%, like this:

Though this could have been caused by the swap size. See comment below.

Solution

  1. We swapped the 400GB HD and replaced it with a 60 GB HD to solve the space issue.

Insallation Process

  1. Downloaded putty, which is a terminal client.
  2. Started debian installation at aprox. 1:45
  3. The partitioning process took around 15 minutes
  4. Installed the base system from 2:20-3:05 (approx 1 hour)
  5. Select and install software 3:07-3:08 (general system installed)
Total time to install Debian onto a Linksys NSLU2: less than two hours (we have a pretty fast network).
